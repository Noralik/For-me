**MysqlARKNIGHT** — ветка, посвящённая созданию и проектированию базы данных, вдохновлённой игрой **Arknights**.  
Здесь собрана логика таблиц, связь сущностей, SQL-структуры и тестовые запросы, которые моделируют игровую механику, персонажей, ресурсы и progression-системы.

---

## 📌 О ветке

Эта ветка включает в себя:

- структуру базы данных (таблицы, ключи, связи);
- описание игровых сущностей Arknights;
- тестовые SQL-запросы;
- эксперименты по оптимизации и нормализации;
- наработки для будущего веб-проекта или API.

Основа проекта — **модель игровых данных**, построенная по принципам реальной RPG-системы.

## 🔍 Основные сущности

### 🧬 **Operators (персонажи)**
- имя, класс, редкость  
- статистика  
- ветка развития  
- навыки и модули  

### 📦 **Resources (материалы)**
- фарм-материалы  
- стоимость крафта  
- редкость  

### 🗺️ **Stages (карты)**
- уровни кампании  
- дроп-таблицы  
- энергия/исчерпаемость  

Все сущности соединены через связи **1:N**, **N:N**, FK и вспомогательные таблицы.

---

## 🛠️ Пример SQL

```sql
CREATE TABLE operators (
    operator_id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50) NOT NULL,
    rarity INT CHECK (rarity BETWEEN 1 AND 6),
    class ENUM('Vanguard','Guard','Defender','Sniper','Caster','Medic','Support','Specialist'),
    description TEXT
);
```
